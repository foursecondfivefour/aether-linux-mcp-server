```json
{
  "action": "action_name",
  "params": {}
}
```

## Prompts

| Prompt | Purpose |
|--------|---------|
| `system_health` | CPU, memory, disk, process, service, and network health report |
| `security_posture` | SELinux/AppArmor, firewall, kernel hardening, Secure Boot, TPM, integrity posture |
| `diagnostic_report` | Journal errors, failed services, OOM events, disk health, network issues, crash hints |

## Resources

AETHER_02 also exposes live `aether://` resources for CPU, memory, disk, OS release, uptime, kernel, GPU, PCI, USB, DMI, thermal, network, firewall, services, users, security, feature gates, environment, installed packages, and recent journal entries.
